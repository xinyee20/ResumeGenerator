# Resume Generator

## Quick start

0. Install [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) in your computer

1. Set up nvm

```shell
nvm install 16.15.0
```

2. Use node 16.15.0 (LTS)

```shell
nvm use 16.15.0
```

3. Install npm package

```shell
npm i
```

4. Start Vite dev server

```shell
npm run dev
```
